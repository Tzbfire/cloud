一、 机器人脚本语言概述
1.1脚本语言介绍
    CodroidV2机器人使用Lua作为机器脚本语言，脚本编程是机器人的底层编程语言，用户可使用脚本生成自定义的命令。
1.2 Lua简介
    Lua 是一种轻量小巧的脚本语言，用标准 C 语言编写并以源代码形式开放，其核心优势在于轻量级、可扩展性（易于与 C/C++ 等语言集成）、灵活的语法以及功能强大的表（table）数据结构，这些特点使得 Lua 常被用于游戏开发（例如在 Unity3D 或 Corona SDK 中作为脚本语言）、嵌入式系统以及科学计算等场景。
1.3 编程环境及配置要求
    脚本编程可在web端脚本界面直接进行编辑和运行。

2.1.6 function(函数)
    在 Lua 中，函数是被看作是"第一类值（First-Class Value）"，函数结构定义如下
optional_function_scope function function_name( argument1, argument2, argument3..., argumentn)
    function_body
    return result_params_comma_separated
end• optional_function_scope: 该参数是可选的指定函数是全局函数还是局部函数，未设置该参数默认为全局函数，如果你需要设置函数为局部函数需要使用关键字 local。
• function_name: 指定函数名称。
• argument1, argument2, argument3..., argumentn: 函数参数，多个参数以逗号隔开，函数也可以不带参数。
• function_body: 函数体，函数中需要执行的代码语句块。
• result_params_comma_separated: 函数返回值，Lua语言函数可以返回多个值，每个值以逗号隔开。
示例1：

    c = a+b
    return c,a,b

string.format()
    生成具有特定格式的字符串, 函数的第一个参数是格式 , 之后是对应格式中每个代号的各种数据
    格式字符串可能包含以下的转义码
• %c - 接受一个数字, 并将其转化为ASCII码表中对应的字符
• %d, %i - 接受一个数字并将其转化为有符号的整数格式
• %o - 接受一个数字并将其转化为八进制数格式
• %x - 接受一个数字并将其转化为十六进制数格式, 使用小写字母
• %X - 接受一个数字并将其转化为十六进制数格式, 使用大写字母
• %e - 接受一个数字并将其转化为科学记数法格式, 使用小写字母e
• %E - 接受一个数字并将其转化为科学记数法格式, 使用大写字母E
• %f - 接受一个数字并将其转化为浮点数格式
• %g(%G) - 接受一个数字并将其转化为%e(%E, 对应%G)及%f中较短的一种格式
• %q - 接受一个字符串并将其转化为可安全被Lua编译器读入的格式
• %s - 接受一个字符串并按照给定的参数格式化该字符串
    为进一步细化格式, 可以在%号后添加参数. 参数将以如下的顺序读入:
• (1) 符号: 一个+号表示其后的数字转义符将让正数显示正号. 默认情况下只有负数显示符号.

    ◦ index(类型:number) - 要读取的字节起始索引，从0开始，小于0时从尾部计算，例如-1表示最后一个字节，-3表示倒数第三个字节
    ◦ type(类型:string) - 表示data按什么类型添加，可选值{Int8, UInt8, Int16, UInt16, Int32, UInt32, Int64, UInt64, Float, Double,String}
    ◦ count(类型:number) - 要读取的指定类型数据的个数，可不传，默认值1，当type为String时，表示要读取的字符串长度
注意：如果读取类型是字符串，读取的字节中有0值会截断字符串，实际返回的字符串会小于读取长度。
返回：res(类型:number/table) - 当count为1时或type为String时，直接返回对应的数字或字符串，当count>1时，会返回数组
3.19.9 追加字节
bw:append(bw1, index, count)
• 功能：将bw1的字节内容按指定索引和长度追加到bw的末尾
• 参数：
    ◦ bw1(类型:ByteWrapper) - 实例化对象
    ◦ index(类型:number) - bw1的起始索引，可不传，默认0，小于0时从尾部计算，例如-1表示最后一个字节，-3表示倒数第三个字节
    ◦ count(类型:number) - 要追加的字节数量，可不传，默认0，当count<=0时，会追加起始索引后的全部字节
3.19.10 复制字节
bw:copy(bw1, targetIndex, srcIndex, count)
• 功能：将bw1从srcIndex开始，复制count字节到bw的targetIndex处
注意：复制不能超过bw当前字节长度，如有需要，先调用resize方法设置长度后复制
• 参数：
    ◦ bw1(类型:ByteWrapper) - 实例化对象
    ◦ targetIndex(类型:number) - bw的起始索引，小于0时从尾部计算，例如-1表示最后一个字节，-3表示倒数第三个字节
    ◦ srcIndex(类型:number) - bw1的起始索引，可不传，默认0，小于0时从尾部计算，例如-1表示最后一个字节，-3表示倒数第三个字节
    ◦ count(类型:number) - 要复制的字节数量，可不传，默认0，当count<=0时，会复制bw1起始索引后的全部字节
3.19.11 打印字节
bw:printByte(index, count, type)
• 功能：打印当前字节数据
• 参数：
    ◦ Index(类型:number) - 要打印的起始索引，可不传，默认值0，小于0时从尾部计算,例如-1表示最后一个字节，-3表示倒数第三个字节
    ◦ count(类型:number) - 打印的字节数，可不传，默认值0，count<=0时打印到结束
    ◦ type(类型:string) - 可不传，默认值h，h表示输出16进制，d表示输出10进制

4.1模块功能介绍
    模块功能是可以调用的已经写好的功能块，对于不同工程都需要调用的功能可以封装成模块，可以使脚本看起来结构更清晰
    客户可以根据需要在模块内封装对象，函数，同时支持传参和返回值
(对象封装)示例：
-- str --
-- 定义类
String = {}

-- 初始化对象
function String:new(str)
  -- 创建一个新的table并作为对象返回，同时进行属性初始化以及赋值
  local newObj = {
    str = str or nil
  }
  -- 设置新对象的元表为 String类，以便继承 String类 的方法
  setmetatable(newObj, self)
  self.__index = self
  -- 返回创建的对象，方便后续调用
  return newObj
end

-- 成员函数（打印）
function String:print()
  print(self.str)
end

-- 成员函数（追加字符串，并返回追加后的字符串）
function String:append(str2)
  self.str = self.str..str2
  return self.str
end
-- add --
-- 分文件成员函数（追加字符串，并返回追加后的字符串）
function String:add(str2)
  self.str = self.str..str2
  return self.str
end-- 工程主函数 -- 
-- 调用模块，注册对象
callModule("string","str")
